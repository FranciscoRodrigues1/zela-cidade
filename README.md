# 🚀API ZelaCidade
## 📌 Sobre o Projeto
A API **ZelaCidade** foi criada para registrar e gerenciar projetos urbanos, como:

- Buracos
- Vazamentos
- Lixo
- Iluminação

Essa API nos permite criar, visualizar, atualizar e deletar ocorrências.

## 🛠️ Tecnologias utilizadas

- Node
- Express
- SQlite
- SQlite3
- Postman
- Nodemon

---
## 📦 Instalação
`npm install`

## ▶️ Como Executar

```bash
npm run dev
```
`http://localhost:3000`

[Clique Aqui](http://lacalhost:3000)
